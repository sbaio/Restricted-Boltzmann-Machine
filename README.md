# Restricted-Boltzmann-Machine

An implementation of RBM totally on Python.

## Requirement 

* Python2.7
* Numpy

## How to Run

```
$python RBM.py --help
```

## Training 

see Hinton's paper ([A Practical Guide to Training Restricted Boltzmann Machines](https://www.cs.toronto.edu/~hinton/absps/guideTR.pdf))
